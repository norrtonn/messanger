Руководство пользователя веб-мессенджера

1. Начало работы

1.1. Регистрация нового аккаунта

Шаг 1: Откройте главную страницу мессенджера
В адресной строке браузера введите: http://localhost:5000
Вы увидите страницу входа в систему

Шаг 2: Перейдите к регистрации
Нажмите на ссылку "Зарегистрироваться" под кнопкой входа
Или перейдите по адресу: http://localhost:5000/register

Шаг 3: Заполните форму регистрации
Важно: Имя пользователя должно быть уникальным. Если такое имя уже существует, система предложит выбрать другое.

Шаг 4: Подтверждение регистрации
Нажмите кнопку "Зарегистрироваться"
При успешной регистрации вы автоматически войдете в систему
Вы увидите сообщение: "Регистрация успешна! Теперь вы можете войти."

1.2. Вход в существующий аккаунт

Шаг 1: Откройте страницу входа
Перейдите по адресу: http://localhost:5000/login

Шаг 2: Введите учетные данные

Шаг 3: Вход в систему
Нажмите кнопку "Войти"
При успешном входе вы увидите главную страницу с вашими чатами
В правом верхнем углу будет отображаться ваше имя пользователя

2. Основные функции мессенджера

2.1. Поиск пользователей
Как найти пользователя для начала общения:

Шаг 1: Используйте поле поиска
Введите имя или часть имени пользователя в поле "Введите имя пользователя..."
Нажмите кнопку "Найти" или клавишу Enter

Шаг 2: Просмотр результатов поиска
Откроется страница с результатами поиска
Каждый найденный пользователь отображается в карточке:

Шаг 3: Начало общения
Нажмите кнопку "Начать чат" под нужным пользователем
Система автоматически создаст приватный чат между вами
Вы перейдете в окно чата с выбранным пользователем
Примечание: Если чат с этим пользователем уже существует, вы попадете в существующий чат с историей сообщений.

2.3. Работа с чатами

2.3.1. Список чатов
На главной странице отображаются все ваши чаты:
Особенности:
Чаты сортируются по времени последнего сообщения (сверху - самые свежие)
Цифра в кружке справа показывает количество непрочитанных сообщений
Нажмите на любой чат, чтобы открыть его

2.3.2. Открытие существующего чата

Шаг 1: Выберите чат из списка
Нажмите на карточку чата с нужным пользователем

Шаг 2: Переход в чат
Вы перейдете в окно чата
Все непрочитанные сообщения автоматически помечаются как прочитанные
Отображается полная история переписки

2.4. Обмен сообщениями

2.4.1. Отправка сообщений
Способы отправки сообщения:

Способ 1: Через поле ввода
Введите текст сообщения в поле "Введите сообщение..."
Нажмите кнопку ↑ справа от поля ввода

Способ 2: Используя клавиатуру
Введите текст сообщения
Нажмите клавишу Enter для отправки
Используйте Shift + Enter для переноса строки
Визуальные особенности:
Ваши сообщения отображаются справа, с синим фоном
Сообщения собеседника отображаются слева, с белым фоном
Под каждым сообщением отображается время отправки (часы:минуты)
Новые сообщения автоматически прокручиваются в область видимости

2.4.2. Автообновление сообщений
Система автоматически проверяет новые сообщения:
Интервал обновления: каждые 3 секунды
Новые сообщения появляются автоматически без перезагрузки страницы
Индикатор загрузки не отображается (фоновая работа)

2.6. Поиск в чате
В данный момент функция недоступна - запланирована в будущих обновлениях.

3. Безопасность и конфиденциальность

3.1. Защита данных
Что защищено:
Пароли хранятся в зашифрованном виде
Сессии защищены от перехвата
Данные передаются по защищенному соединению (если используется HTTPS)
Что НЕ защищено:
Сообщения не шифруются в базе данных
Нет сквозного шифрования
Администратор системы может просматривать все сообщения

3.2. Рекомендации по безопасности
Используйте сложные пароли: минимум 8 символов, буквы + цифры
Не передавайте пароли: даже администраторам системы
Выходите из системы: при использовании на общественных компьютерах
Проверяйте URL: убедитесь, что находитесь на правильном сайте

4. Устранение неполадок
4.1. Распространенные проблемы и решения
Проблема: "Не могу войти в систему"
Возможные причины и решения:
Неверное имя пользователя или пароль
Проверьте правильность ввода (регистр имеет значение)
Используйте функцию восстановления пароля (если доступна)
Аккаунт не существует
Зарегистрируйте новый аккаунт
Убедитесь, что вы правильно помните имя пользователя
Проблемы с браузером
Очистите кэш браузера (Ctrl+Shift+Del)
Попробуйте другой браузер
Проверьте, включены ли cookies
Проблема: "Сообщения не отправляются"
Решение:
Проверьте подключение к интернету
Обновите страницу (F5 или Ctrl+R)
Проверьте, не пустое ли сообщение
Убедитесь, что поле ввода активно (кликните по нему)
Проблема: "Не вижу новых сообщений"
Решение:
Подождите 3-5 секунд (автообновление срабатывает каждые 3 секунды)
Обновите страницу вручную (F5)
Проверьте, открыт ли правильный чат
Убедитесь, что собеседник действительно отправил сообщение
Проблема: "Не могу найти пользователя"
Решение:
Убедитесь, что пользователь зарегистрирован в системе
Проверьте правильность написания имени
Попробуйте поиск по части имени
Убедитесь, что это не ваш собственный аккаунт

4.2. Коды ошибок
Ошибка 404: Страница не найдена
Проверьте правильность URL
Вернитесь на главную страницу
Ошибка 500: Внутренняя ошибка сервера
Обновите страницу
Попробуйте позже
Сообщите администратору

5. Использование на мобильных устройствах
5.1. Адаптивный дизайн
Особенности мобильной версии:
Упрощенная навигация:
Кнопки увеличены для удобного нажатия
Меньше элементов на экране
Оптимизированные отступы
Удобный ввод сообщений:
Клавиатура автоматически появляется при фокусе на поле ввода
Кнопка отправки доступна рядом с клавиатурой
Оптимизированный просмотр чатов:
Список чатов занимает весь экран
Легкое переключение между чатами свайпом

5.2. Рекомендации для мобильных устройств
Добавьте на главный экран:
Откройте мессенджер в браузере
В меню браузера выберите "Добавить на главный экран"
Получите иконку как у обычного приложения
Используйте темную тему (если доступно):
Включите темную тему в настройках браузера
Сохраните заряд батареи

6. Часто задаваемые вопросы (FAQ)
Q1: Можно ли восстановить удаленные сообщения?
A: Нет, в текущей версии функция удаления сообщений не реализована, поэтому и восстановление не требуется.
Q2: Поддерживаются ли групповые чаты?
A: Нет, в текущей версии доступны только приватные чаты между двумя пользователями.
Q3: Можно ли отправлять файлы или изображения?
A: Нет, в текущей версии поддерживается только текстовый обмен сообщениями.
Q4: Есть ли мобильное приложение?
A: Нет, но веб-версия полностью адаптирована для мобильных устройств и может быть добавлена на главный экран.
Q5: Как долго хранятся сообщения?
A: Сообщения хранятся бессрочно в базе данных системы.
Q6: Можно ли изменить имя пользователя?
A: Нет, в текущей версии изменение имени пользователя не предусмотрено.
Q7: Поддерживаются ли уведомления?
A: Да, есть уведомления в заголовке вкладки браузера о количестве непрочитанных сообщений.

7. Контакты и поддержка

7.1. Получение помощи
Если у вас возникли проблемы:
Обратитесь к администратору системы
Проверьте раздел "Устранение неполадок" в этом руководстве
Обновите браузер до последней версии

7.2. Сообщение об ошибках
При сообщении об ошибке укажите:
Описание проблемы: что произошло и что вы ожидали
Шаги воспроизведения: как вызвать ошибку
Скриншот или видео: если возможно
Информация о системе: браузер, ОС, устройство
Время возникновения: когда произошла ошибка

Краткая памятка пользователя
Быстрый старт:
Регистрация → Введите имя и пароль
Поиск пользователя → Введите имя в поиске
Начало чата → Нажмите "Начать чат"
Общение → Пишите сообщения и нажимайте Enter
Горячие клавиши:
Enter → Отправить сообщение
Shift + Enter → Перенос строки
F5 → Обновить страницу
Ctrl + R → Обновить страницу (Mac: Cmd + R)
Важные моменты:
 Сообщения сохраняются автоматически
 Новые сообщения обновляются каждые 3 секунды
 Чат создается автоматически при первом сообщении
 Выходите из системы на общественных компьютерах
